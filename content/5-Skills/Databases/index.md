### Database Systems
- **SQL Server:** Relational database design and enterprise data management.
- **MySQL:** Database development and administration.
- **SQLite:** Embedded database applications.
- **Redis:** In-memory caching and high-performance data storage.

### Database Skills
- **Database Design:** Schema design, normalization, and indexing.
- **SQL:** Query writing, optimization, joins, aggregations, and transactions.
- **Data Processing:** Large-scale data manipulation and ETL workflows.
