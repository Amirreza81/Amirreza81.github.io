### Programming Languages
- **Python:** Primary language for AI, machine learning, data science, automation, and backend development.
- **Java:** Object-oriented programming, desktop applications, networking, and software engineering.
- **C:** Low-level programming, algorithms, memory management, and systems programming.
- **SQL:** Database querying, management, and analytical data processing.
- **R:** Statistical computing and data visualization.

### Web Technologies
- **HTML & CSS:** Responsive web development and UI customization.

### AI & Data Science Libraries
- **PyTorch:** Deep learning model development and research.
- **TensorFlow:** Machine learning and deep learning applications.
- **Scikit-learn:** Classical machine learning, preprocessing, and model evaluation.
- **OpenCV:** Computer vision, image processing, and video analysis.
- **NumPy:** Numerical computing and scientific programming.
- **Pandas:** Data manipulation, cleaning, and analysis.
- **Matplotlib:** Data visualization and analytical plotting.
